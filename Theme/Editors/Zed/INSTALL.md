# Installing Steelbore Theme for Zed Editor

**Version:** 1.0 | **Author:** Mohamed Hammd | **License:** GPL3+ | **Website:** [steelbore.com](https://steelbore.com)

## Installation

1. Copy `steelbore.json` to your Zed themes directory:
   - **macOS/Linux:** `~/.config/zed/themes/`
   - **Windows:** `%APPDATA%\Zed\themes\`
2. Open Zed → **Command Palette** (`Ctrl+Shift+P`) → **theme selector: Toggle**.
3. Select **Steelbore** from the list.

## Alternative: Settings File

Add to your Zed `settings.json`:
```json
{
    "theme": "Steelbore"
}
```
