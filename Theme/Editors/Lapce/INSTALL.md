# Installing Steelbore Theme for Lapce Editor

**Version:** 1.0 | **Author:** Mohamed Hammd | **License:** GPL3+ | **Website:** [steelbore.com](https://steelbore.com)

## Installation

1. Copy `steelbore.toml` to your Lapce themes directory:
   - **Linux:** `~/.config/lapce-stable/themes/`
   - **macOS:** `~/Library/Application Support/dev.lapce.Lapce-Stable/themes/`
   - **Windows:** `%APPDATA%\dev.lapce.Lapce-Stable\themes\`
2. Open Lapce → **Command Palette** (`Ctrl+Shift+P`) → **Change Color Theme**.
3. Select **Steelbore** from the list.
