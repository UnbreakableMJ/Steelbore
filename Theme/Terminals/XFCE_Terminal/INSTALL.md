# Installing Steelbore Theme for XFCE Terminal

**Version:** 1.0 | **Author:** Mohamed Hammd | **License:** GPL3+ | **Website:** [steelbore.com](https://steelbore.com)

## Installation

1. Copy `Steelbore.theme` to:
   - `~/.local/share/xfce4/terminal/colorschemes/`
   - Or system-wide: `/usr/share/xfce4/terminal/colorschemes/`
2. Open XFCE Terminal → **Edit** → **Preferences** → **Colors**.
3. Under **Presets**, select **Steelbore**.
