# Installing Steelbore Theme for WezTerm

**Version:** 1.0 | **Author:** Mohamed Hammd | **License:** GPL3+ | **Website:** [steelbore.com](https://steelbore.com)

## Installation

1. Copy `steelbore.lua` to your WezTerm colors directory:
   - `~/.config/wezterm/colors/`
2. In your `~/.config/wezterm/wezterm.lua`, add:
   ```lua
   config.color_scheme = "steelbore"
   ```
3. Save and WezTerm will reload automatically.
