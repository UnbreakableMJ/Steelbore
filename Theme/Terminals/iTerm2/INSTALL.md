# Installing Steelbore Theme for iTerm2

**Version:** 1.0 | **Author:** Mohamed Hammd | **License:** GPL3+ | **Website:** [steelbore.com](https://steelbore.com)

## Installation

1. Double-click `Steelbore.itermcolors` — iTerm2 will import it automatically.
2. *Or* go to **iTerm2** → **Preferences** → **Profiles** → **Colors** → **Color Presets…** → **Import…** → select `Steelbore.itermcolors`.
3. Select **Steelbore** from the Color Presets dropdown.
