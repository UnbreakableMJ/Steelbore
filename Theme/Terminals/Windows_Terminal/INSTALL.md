# Installing Steelbore Theme for Windows Terminal

**Version:** 1.0 | **Author:** Mohamed Hammd | **License:** GPL3+ | **Website:** [steelbore.com](https://steelbore.com)

## Installation

1. Open Windows Terminal → **Settings** (or press `Ctrl+,`).
2. Click **Open JSON file** (bottom-left corner).
3. In the `"schemes"` array, paste the contents of `steelbore.json`.
4. Under the profile you want to theme, set:
   ```json
   "colorScheme": "Steelbore"
   ```
5. Save and close the file. The theme is applied instantly.
