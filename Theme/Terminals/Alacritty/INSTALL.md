# Installing Steelbore Theme for Alacritty

**Version:** 1.0 | **Author:** Mohamed Hammd | **License:** GPL3+ | **Website:** [steelbore.com](https://steelbore.com)

## Installation

1. Copy `steelbore.toml` to your Alacritty config directory:
   - **Linux:** `~/.config/alacritty/themes/`
   - **macOS:** `~/.config/alacritty/themes/`
   - **Windows:** `%APPDATA%\alacritty\themes\`
2. In your main `alacritty.toml`, add:
   ```toml
   import = ["~/.config/alacritty/themes/steelbore.toml"]
   ```
3. Restart Alacritty.
