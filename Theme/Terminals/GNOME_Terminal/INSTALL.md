# Installing Steelbore Theme for GNOME Terminal

**Version:** 1.0 | **Author:** Mohamed Hammd | **License:** GPL3+ | **Website:** [steelbore.com](https://steelbore.com)

## Installation

1. Make the script executable:
   ```bash
   chmod +x steelbore-gnome-terminal.sh
   ```
2. Run the script:
   ```bash
   ./steelbore-gnome-terminal.sh
   ```
3. Open GNOME Terminal → **Preferences** → **Profiles** → select **Steelbore**.
