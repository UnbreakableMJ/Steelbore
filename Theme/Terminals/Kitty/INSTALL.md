# Installing Steelbore Theme for Kitty Terminal

**Version:** 1.0 | **Author:** Mohamed Hammd | **License:** GPL3+ | **Website:** [steelbore.com](https://steelbore.com)

## Installation

1. Copy `steelbore.conf` to your Kitty config directory:
   - `~/.config/kitty/themes/`
2. In your `~/.config/kitty/kitty.conf`, add:
   ```
   include themes/steelbore.conf
   ```
3. Restart Kitty (or press `Ctrl+Shift+F5`).
