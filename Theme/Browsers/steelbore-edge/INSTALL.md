# Installing Steelbore Theme for Microsoft Edge

**Version:** 1.0 | **Author:** Mohamed Hammd | **License:** GPL3+ | **Website:** [steelbore.com](https://steelbore.com)

## Method 1: Load as Unpacked Extension (Developer)

1. Open Edge.
2. Navigate to `edge://extensions`.
3. Enable **Developer mode** (toggle in the bottom-left corner).
4. Click **Load unpacked**.
5. Select the `steelbore-edge/` folder.
6. The theme will be applied immediately.

## Method 2: Drag and Drop ZIP

1. Open Edge and navigate to `edge://extensions`.
2. Enable **Developer mode**.
3. Drag and drop the `steelbore-edge-theme.zip` file onto the page.

## Method 3: Edge Add-ons Store

1. Package the theme using Edge's developer tools.
2. Upload to the [Edge Add-ons Developer Dashboard](https://partner.microsoft.com/dashboard/microsoftedge).
3. Once approved, users can install directly from the store.
