# Steelbore Browser Themes - Installation Guide

This directory contains the Steelbore theme for various browsers.

## 🌐 Google Chrome & Microsoft Edge

1. Open Chrome or Edge and navigate to `chrome://extensions` or `edge://extensions`.
2. Enable **Developer mode** (usually a toggle in the top right or sidebar).
3. Click **Load unpacked**.
4. Select the `steelbore-chrome` or `steelbore-edge` folder within this directory.
5. The theme will be applied immediately.

*Note: For official store upload, zip the contents of the respective folder (manifest.json, icon.png, background.png).*

## 🦊 Mozilla Firefox

1. Open Firefox and navigate to `about:debugging#/runtime/this-firefox`.
2. Click **Load Temporary Add-on...**.
3. Select the `manifest.json` file inside the `steelbore-firefox` folder.
4. The theme will be applied immediately.

*Note: To install permanently, the theme must be signed via [AMO (addons.mozilla.org)](https://addons.mozilla.org/).*
