# Steelbore Browser Themes — Installation Guide

**Version:** 1.0 | **Author:** Mohamed Hammd | **License:** GPL3+ | **Website:** [steelbore.com](https://steelbore.com)

This directory contains Steelbore themes for all supported browsers. Each subfolder has its own `INSTALL.md` with detailed instructions.

---

## 🌐 Chromium-Based Browsers

### Google Chrome (`steelbore-chrome/`)

1. Open Chrome → `chrome://extensions` → Enable **Developer mode**.
2. Click **Load unpacked** → select the `steelbore-chrome/` folder.
3. *Or* drag `steelbore-chrome-theme.zip` onto the extensions page.

### Microsoft Edge (`steelbore-edge/`)

1. Open Edge → `edge://extensions` → Enable **Developer mode**.
2. Click **Load unpacked** → select the `steelbore-edge/` folder.
3. *Or* drag `steelbore-edge-theme.zip` onto the extensions page.

### Brave (`steelbore-brave/`)

1. Open Brave → `brave://extensions` → Enable **Developer mode**.
2. Click **Load unpacked** → select the `steelbore-brave/` folder.
3. *Or* drag `steelbore-brave-theme.zip` onto the extensions page.

### Trivalent (`steelbore-trivalent/`)

1. Open Trivalent → `chrome://extensions` → Enable **Developer mode**.
2. Click **Load unpacked** → select the `steelbore-trivalent/` folder.
3. *Or* drag `steelbore-trivalent-theme.zip` onto the extensions page.

### Arc (`steelbore-arc/`)

1. Open Arc → `arc://extensions` → Enable **Developer mode**.
2. Click **Load unpacked** → select the `steelbore-arc/` folder.
3. *Alternative:* Right-click a Space in the sidebar → **Edit Space** → set accent to `#FE6B00` and use `background.png` as wallpaper.

## 🦊 Firefox-Based Browsers

### Mozilla Firefox (`steelbore-firefox/`)

1. Open Firefox → `about:addons` (`Ctrl+Shift+A`).
2. Gear icon ⚙️ → **Install Add-on From File…** → select `steelbore-firefox-theme.xpi`.
3. *Or* temporary load via `about:debugging#/runtime/this-firefox` → **Load Temporary Add-on** → select `manifest.json`.

### Zen Browser (`steelbore-zen/`)

1. Open Zen → `about:addons` (`Ctrl+Shift+A`).
2. Gear icon ⚙️ → **Install Add-on From File…** → select `steelbore-zen-theme.xpi`.
3. *Or* use Zen's native theme settings with accent `#FE6B00` and `background.png`.

### Tor Browser (`steelbore-tor/`)

1. Open Tor Browser → `about:addons` (`Ctrl+Shift+A`).
2. Gear icon ⚙️ → **Install Add-on From File…** → select `steelbore-tor-theme.xpi`.
3. *Note:* This theme changes only visuals — it does not affect Tor's privacy/security features.

## 🎮 Opera Browsers

### Opera GX (`steelbore-opera-gx/`)

1. Open Opera GX → `opera://extensions` → Enable **Developer mode**.
2. Click **Load unpacked** → select the `steelbore-opera-gx/` folder.
3. *Or* drag `steelbore-opera-gx.zip` onto the extensions page.
4. *Store upload:* Upload to [GX.store](https://store.gx.me/) as a GX Mod.

### Opera One / Opera Air (`steelbore-opera-one/`)

1. Open Opera One → `opera://extensions` → Enable **Developer mode**.
2. Click **Load unpacked** → select the `steelbore-opera-one/` folder.
3. *Alternative:* Use **Easy Setup** → **Classic** theme → add `background.png` as wallpaper and adjust colors to cool/dark.

### Opera Neon (`steelbore-opera-neon/`)

Opera Neon does not support theme extensions. Apply manually:

1. Go to **Settings** → **Customization** → **Wallpapers** → add `background.png`.
2. Set accent color to `#FE6B00` (Steel Orange).
3. See `steelbore-opera-neon/INSTALL.md` for full color reference table.
