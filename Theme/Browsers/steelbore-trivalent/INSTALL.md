# Installing Steelbore Theme for Trivalent Browser

**Version:** 1.0 | **Author:** Mohamed Hammd | **License:** GPL3+ | **Website:** [steelbore.com](https://steelbore.com)

## Method 1: Load as Unpacked Extension (Developer)

1. Open Trivalent.
2. Navigate to `chrome://extensions`.
3. Enable **Developer mode** (toggle in the top-right corner).
4. Click **Load unpacked**.
5. Select the `steelbore-trivalent/` folder.
6. The theme will be applied immediately.

## Method 2: Drag and Drop ZIP

1. Open Trivalent and navigate to `chrome://extensions`.
2. Enable **Developer mode**.
3. Drag and drop the `steelbore-trivalent-theme.zip` file onto the page.

## Notes

- Trivalent is a Chromium-based, security-focused browser and fully supports Chrome theme extensions.
- This theme changes only the visual appearance — it does not affect Trivalent's security hardening.
