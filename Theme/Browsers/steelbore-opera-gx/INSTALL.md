# Installing Steelbore Mod for Opera GX

**Version:** 1.0 | **Author:** Mohamed Hammd | **License:** GPL3+ | **Website:** [steelbore.com](https://steelbore.com)

## Method 1: Load as Unpacked Mod (Recommended)

1. Open Opera GX.
2. Navigate to `opera://extensions`.
3. Enable **Developer mode** (toggle in the top-right corner).
4. Click **Load unpacked**.
5. Select the `steelbore-opera-gx/` folder.
6. The Steelbore mod will appear in your mod list and can be activated.

## Method 2: Load from ZIP

1. Open Opera GX.
2. Navigate to `opera://extensions`.
3. Enable **Developer mode**.
4. Drag and drop the `steelbore-opera-gx.zip` file onto the extensions page.

## Method 3: Upload to GX Mod Store

1. Visit [GX.store](https://store.gx.me/) to create a developer account.
2. Upload the contents of the `steelbore-opera-gx/` folder as a new Mod.
3. Once approved, users can install it directly from the GX Mod Store.
