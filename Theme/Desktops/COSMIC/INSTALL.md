# Installing Steelbore Theme for COSMIC Desktop

**Version:** 1.0 | **Author:** Mohamed Hammd | **License:** GPL3+ | **Website:** [steelbore.com](https://steelbore.com)

## Installation

1. Copy `steelbore.ron` to:
   - `~/.config/cosmic/com.system76.CosmicTheme.Dark/v1/`
2. Copy `Steelbore_wallpaper_blue.png` to your wallpapers directory.
3. Open **Settings** → **Desktop** → **Appearance** → select the Steelbore theme.
4. Set wallpaper via **Settings** → **Desktop** → **Wallpaper**.
