# Installing Steelbore Theme for LeftWM

**Version:** 1.0 | **Author:** Mohamed Hammd | **License:** GPL3+ | **Website:** [steelbore.com](https://steelbore.com)

## Installation

1. Copy `theme.toml` and `polybar.config` to:
   - `~/.config/leftwm/themes/steelbore/`
2. Copy `Steelbore_wallpaper_blue.png` to the same directory.
3. Symlink as the current theme:
   ```bash
   ln -sf ~/.config/leftwm/themes/steelbore ~/.config/leftwm/themes/current
   ```
4. Restart LeftWM (`Mod+Shift+R`).
