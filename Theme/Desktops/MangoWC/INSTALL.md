# Installing Steelbore Theme for MangoWC

**Version:** 1.0 | **Author:** Mohamed Hammd | **License:** GPL3+ | **Website:** [steelbore.com](https://steelbore.com)

## Installation

1. Copy `theme.toml` to your MangoWC config directory:
   - `~/.config/mangowc/themes/steelbore.toml`
2. In your MangoWC config, set:
   ```toml
   theme = "steelbore"
   ```
3. Copy `Steelbore_wallpaper_blue.png` to your wallpapers directory.
4. Restart MangoWC.
