# Installing Steelbore Theme for MATE Desktop

**Version:** 1.0 | **Author:** Mohamed Hammd | **License:** GPL3+ | **Website:** [steelbore.com](https://steelbore.com)

## Installation

### Window Borders (Metacity)
1. Create the theme directories:
   ```bash
   mkdir -p ~/.themes/Steelbore/metacity-1/
   ```
2. Copy `metacity-theme-1.xml` to:
   - `~/.themes/Steelbore/metacity-1/`
3. Open **System** → **Preferences** → **Appearance** → **Customize** → **Window Border** → select **Steelbore**.

### Wallpaper
1. Copy `Steelbore_wallpaper_blue.png` to `~/Pictures/`.
2. Right-click desktop → **Change Desktop Background** → select the wallpaper.
