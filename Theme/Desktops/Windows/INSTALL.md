# Installing Steelbore Theme for Windows

**Version:** 1.0 | **Author:** Mohamed Hammd | **License:** GPL3+ | **Website:** [steelbore.com](https://steelbore.com)

## Installation

### Windows Theme
1. Copy `Steelbore.theme` to `C:\Windows\Resources\Themes\`.
2. Copy `Steelbore_wallpaper_blue.png` to the same directory.
3. Right-click desktop → **Personalize** → select **Steelbore** under Themes.

### Dark Mode
1. Go to **Settings** → **Personalization** → **Colors**.
2. Set to **Dark** mode.
3. Set accent color to a custom color: `#FE6B00` (Steel Orange).

### Windows Terminal
See `Theme/Terminals/Windows_Terminal/INSTALL.md` for terminal color scheme.
