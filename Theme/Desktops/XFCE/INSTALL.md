# Installing Steelbore Theme for XFCE

**Version:** 1.0 | **Author:** Mohamed Hammd | **License:** GPL3+ | **Website:** [steelbore.com](https://steelbore.com)

## Installation

### GTK Theme
1. Create the theme directory:
   ```bash
   mkdir -p ~/.themes/Steelbore/gtk-3.0/
   ```
2. Copy `gtk.css` to:
   - `~/.themes/Steelbore/gtk-3.0/gtk.css`
3. Open **Settings** → **Appearance** → select **Steelbore**.

### Wallpaper
1. Copy `Steelbore_wallpaper_blue.png` to `~/Pictures/`.
2. Right-click desktop → **Desktop Settings** → select the wallpaper.

### XFCE Terminal
See `Theme/Terminals/XFCE_Terminal/INSTALL.md` for terminal color scheme.
