# Steelbore (Official)

The definitive Steelbore theme. A premium dark experience featuring deep navy backgrounds and high-visibility orange accents.

## Color Palette

- **Background**: `#0E141D` - Primary backdrop
- **Foreground**: `#F0F0F0` - Elegant text
- **Steel Orange**: `#FE6B00` - Key interactions and emphasis
- **Steel Blue**: `#3B82F6` - Functional accents
- **Steel Green**: `#10B981` - Success states and secondary details

## Typography

- **Headers**: Outfit Bold
- **Body Text**: Roboto Regular

## Best Used For

Code editors, system terminals, professional development environments, and any interface requiring focus and modern sophistication.
